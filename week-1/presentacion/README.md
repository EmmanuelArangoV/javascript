# Explorador Interactivo de Arrays Avanzados en JavaScript

Este es un pequeño proyecto interactivo para demostrar y explorar métodos avanzados de arrays en JavaScript, siguiendo un enfoque de programación declarativa.

## Temas Cubiertos

-   **Métodos de Array Fundamentales:**
    -   `map`: Para transformar cada elemento de un array.
    -   `filter`: Para seleccionar elementos que cumplen una condición.
    -   `reduce`: Para reducir un array a un solo valor.
-   **Encadenamiento de Métodos:** Cómo combinar `map`, `filter` y `reduce` para crear pipelines de datos eficientes.
-   **Inmutabilidad:** La importancia de no modificar los datos originales y en su lugar crear nuevas estructuras de datos.
-   **Transformación de Datos de API:** Un ejemplo práctico de cómo procesar datos obtenidos de una API real (`TheCatAPI`).

## Archivos del Proyecto

-   `index.html`: La estructura de la página web.
-   `styles.css`: Los estilos para hacer la presentación visualmente atractiva.
-   `app.js`: Contiene todo el código JavaScript que maneja la **interactividad de la página**. Controla las pestañas, ejecuta los ejemplos y muestra los resultados en la interfaz de usuario.
-   `script.js`: Contiene **ejemplos "crudos"** de los mismos conceptos, diseñados para ser ejecutados directamente en la consola del navegador. Es útil si quieres ver la lógica sin la capa de la interfaz visual.

## Cómo Usarlo

1.  Abre el archivo `index.html` en tu navegador web o visita la [página en vivo](http://emmanuelarangov.github.io/js-presentation/).
2.  Navega por las diferentes pestañas (`MAP`, `FILTER`, `REDUCE`, etc.).
3.  Haz clic en los botones "Ejecutar Ejemplo" para ver el código en acción.
4.  Abre la consola del desarrollador (F12) para ver los `console.log` de `script.js` si lo deseas.
